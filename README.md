# HR Analytics Dashboard
![HR Dashboard](https://github.com/deepikakohli0901-svg/HR-Analytics-Dashboard/blob/main/HR%20analystics%20dashboard%20image%20.png?raw=true)
## Project Overview

This project is an interactive HR Analytics Dashboard developed using Power BI and Excel.

The dashboard provides insights into:

* Employee Attrition
* Gender Distribution
* Department-wise Employees
* Average Salary Analysis
* Average Experience Analysis

## Tools Used

* Power BI
* Microsoft Excel

## Dashboard Features

* Total Employees KPI
* Attrition Analysis
* Gender Ratio Visualization
* Department-wise Workforce Analysis
* Average Salary by Department
* Average Experience by Department
* Interactive Department Filter
